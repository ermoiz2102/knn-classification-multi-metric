# 🤖 K-Nearest Neighbors (KNN) Classification

## 📌 Overview

This project demonstrates KNN classification using both manual implementation and Scikit-learn. It explores multiple distance metrics and combines predictions to improve accuracy.

---

## 🎯 Objective

* Understand KNN algorithm step-by-step
* Implement distance calculations manually
* Compare multiple distance metrics
* Improve prediction using ensemble approach

---

## 📂 Dataset

* Custom dataset (house classification)
* CSV dataset (height & weight classification)

---

## 🔍 Steps Performed

* Manual KNN Implementation
* Distance Calculation (Manhattan)
* Neighbor Selection
* Prediction using Mode
* KNN using Scikit-learn
* Multi-Metric Analysis
* Ensemble Prediction

---

## 📊 Results

* Manual implementation helped understand algorithm fundamentals
* Different distance metrics produced slightly different predictions
* Combining predictions improved robustness and consistency
